Node.js Overview

  • Node js allows developers to create both front-end and back-end applications using JavaScript
  • Node.js is open source which means that the source code for Node.js is publicy available. And its maintained by contributers from all over the world
  • Node.js is cross-platform, so its not dependant on any operating system
  • Node.js is a JavaScript runtime environment whcih when you write JavaScript code in your text editio, that code cannot perform any task unless you execute or run it and to run your code you need a runtime environemnt
  • Browsers like Chrome and Firefox have runtime environments
  • Before Node.js JavaScript could only be run in a browser and it was used to build only front-end applications
  • Node.js platform provides a runtime environemnt outside of the browser.
  • This makes it possible to build back end application usin the dame JavaScript programming language you may be familiar with
